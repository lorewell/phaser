# 工作记忆

## 项目：Phaser擂台对战游戏

### 技术栈
- Phaser 3 (前端游戏引擎)
- Vue 3 + Colyseus (另一个项目，聊天平台)

### 项目结构
- `game.js` - 游戏主逻辑
- `attacks.js` - 攻击系统模块（麻将子弹等）
- `characters.js` - 角色配置
- `menu.js` - 菜单场景
- `characterSelect.js` - 角色选择场景

### 角色攻击类型
- MELEE - 近战（拳拳到肉）
- RANGED - 远程（射子弹）
- CHARGE - 冲锋（影刃）
- AURA - 光环（铁壁反伤）

### 麻将攻击（wuzhuzhu）
- 使用程序化绘制，无需图片
- 34种麻将牌：万子/筒子/条子/字牌
- 不同颜色区分类型

### 用户偏好
- 中文沟通
- 代码模块化、关注点分离
- 功能优先，可用注释预留扩展
