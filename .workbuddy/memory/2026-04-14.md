## v2.0 - 双攻击系统实现（16:22）

### 新增功能
- **角色重构**：Fighter 类，角色由头/身体/手臂/腿部独立图形组成
- **拳击（Punch）**：J / Num1，快速、低伤(7)、短范围(65)、冷却280ms，手臂伸出动画
- **踢击（Kick）**：K / Num2，慢速、高伤(13)、长范围(85)、冷却480ms，抬腿踢出动画
- **攻击状态机**：startup → active → recovery 三阶段，各阶段有独立动画
- **改进防御**：F键防御，格挡仅受15%伤害，星光特效
- **受击效果**：身体倾斜动画 + 白色闪光 + 差异化伤害数字颜色
- **场景美化**：霓虹紫色网格背景 + 地面发光线

### 操作说明
- P1: WASD移动, J拳, K踢, F防御
- P2: 方向键移动, Num1拳, Num2踢, Num0防御

### 设计参数
- 拳: 伤害7, 范围65, 冷却280ms
- 踢: 伤害13, 范围85, 冷却480ms
