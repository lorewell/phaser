# Pixel Chess - 像素象棋

基于 Phaser 3 引擎自制的复古像素风格中国象棋。

## 游戏功能
- **核心玩法**：完整的中国象棋逻辑，包含所有棋子的移动规则（马蹩脚、炮隔山打牛等）及吃子判定。
- **对战模式**：
  - **本地双人对战**：在一台设备上轮渡进行红、蓝棋子的对抗。
  - **玩家对阵电脑**：玩家执红对抗 AI，AI 基于 Alpha-Beta 剪枝搜索实现。
  - **游戏设置**：提供简单、普通、困难三档难度调节。
- **特色机制**：
  - **动态像素生成**：游戏内所有棋子和棋盘均通过代码实时绘制，无需外部图片资源，保持极致像素颗粒感。
  - **悔棋系统**：支持无限次撤回操作（PVE 模式下点击一次悔棋将回退双方各一步）。
  - **交互反馈**：选中棋子高亮显示、合法走点（绿点）提示，以及走棋时的平滑位移。

## 代码结构
- `index.html`: 入口文件，加载 Phaser 框架及所有逻辑脚本。
- `js/config.js`: 全局物理、画布及场景配置，核心渲染开启了 `pixelArt: true` 以保持清晰。
- `js/Logic.js`: **核心引擎**。定义了棋子类及各自的移动规则逻辑映射，维护棋局历史步数。
- `js/AI.js`: **AI 核心**。包含局面评估函数及多层深度搜索算法。
- `js/MenuScene.js`: 游戏开始菜单场景，处理模式选择与难度切换。
- `js/GameScene.js`: **游戏交互与绘制场景**。负责棋子的生成、点击交互、动画播放以及胜负逻辑判定。

## 如何运行
直接在支持 HTML5 的现代浏览器中打开 `index.html` 即可开始对局。
