# MEMORY.md - 《砖块幸存者》项目长期记忆

## 项目基本信息
- **项目路径**：`d:/lw/phaser`
- **游戏类型**：Roguelike + 打砖块，Phaser 3，纯 JS ES Module
- **画布**：480×800 竖屏，`Phaser.Scale.FIT`
- **运行方式**：需本地 HTTP 服务器（VSCode Live Server 等）

## 文件结构
```
js/
  main.js          # Phaser 配置 + 场景注册
  settings.js      # 设置管理（localStorage + 球速/难度常量）
  scenes/
    MenuScene.js   # 主菜单
    SettingsScene.js
    GameScene.js   # 游戏主逻辑（核心，v0.3版本）
```

## 当前版本状态（v0.3，2026-04-13）
### 三条隐性流派
| 流派 | 核心Buff | 联动机制 |
|------|---------|---------|
| 多球系 | 混沌核心(25%分裂/叠)、虫洞弹射(+20%新球速)、磁场共振(球≥3时伤害×2) |
| 激光系 | 激光炮台(每4s激光/可缩到1.5s)、等离子穿透(穿盾)、超载放电(溅射1格) |
| 强击系 | 穿甲弹头(+1伤害/可叠5)、超频驱动(球速×1.15/可叠4)、反弹加速(+8%/每次弹板)、磁力延展(+50宽) |
| 通用 | 爆裂连锁(爆炸范围扩大)、过载协议(每20块额外一次抽取) |

### 圣遗物系统（砖块小概率掉落，永久生效）
- 时间碎片: 漏球不扣血，每波一次
- 能量核心: 每波开始炸2块砖
- 虚空裂隙: 15%概率无视碰撞
- 死亡契约: 初始血-1，但每波三选四
- 引力透镜: 球靠近挡板时轻微吸引

### 特殊事件波次
- 波次5「精英波」: 全场HP×2，清完给圣遗物
- 波次10「Boss波」: 40HP移动Boss砖块，击败掉落稀有圣遗物

### 核心数值
- 球基础伤害: 1 (+穿甲弹头叠加)
- 混沌分裂: 25%概率/叠加，最多5球
- 激光穿透: 需先有激光炮台
- 过载协议: 每20块额外一次三选N抽取机会
